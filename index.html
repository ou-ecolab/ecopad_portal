    End to End Demo

    This is a minimal example for the interaction of the various components of EcoPAD. The components are:

    • The frontend (the webpage you are looking at) - consisting of html, javascript and bootstrap (a CSS,javascript framework).
    • The cybercommons framework providing a RESTful API by means of a microservice framework (django, celery, mongodb, nginx)
    • A task repository, containing python code to be consumed by the celery worker connected to the above mentioned celery application
    • A docker container for the actual computation which is implemented in Fortran. This container receives commands from the celery worker.
